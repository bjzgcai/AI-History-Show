# PROLOG

## Full Name
The Birth of Prolog

## Year / Period
1973

## Type
Language

## One-Sentence Summary
PROLOG is a major AI100 achievement connecting Language to later AI systems.

## Hero Description
PROLOG made logic executable. Instead of writing step-by-step procedures, programmers wrote facts and rules, then asked queries that the system solved by unification and backtracking. The language joined theorem proving, natural-language work, and knowledge representation into a practical symbolic AI tool.

## People & Place
- Alain Colmerauer: Creator of the first Prolog implementation
- Robert Kowalski: Logic programming theorist

Key place: Aix-Marseille University and University of Edinburgh

## Historical Background
By the early 1970s, symbolic AI needed languages that could express rules, relations, and search without burying them under procedural code.

## Canonical Source
The Birth of Prolog. Alain Colmerauer and Philippe Roussel, ACM SIGPLAN Notices, 1993. https://doi.org/10.1145/155360.155362

## Core Idea
A Prolog program is a database of relations. Query evaluation becomes a controlled search through logical consequences.

## Key Concepts
- Facts: Base relations such as parent(alice,bob) define the world the program knows.
- Rules: Rules describe how new relations follow from existing ones.
- Backtracking: When one path fails, Prolog returns to a choice point and tries another proof.

## Impact
Prolog influenced expert systems, computational linguistics, constraint logic programming, and modern declarative programming ideas.
