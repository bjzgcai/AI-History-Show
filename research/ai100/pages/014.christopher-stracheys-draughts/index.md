# Christopher Strachey's Draughts

## Full Name
Christopher Strachey's Draughts

## Year / Period
1951-1952

## Type
Game AI

## One-Sentence Summary
Strachey's draughts program is an AI100 achievement in early game AI.

## Hero Description
Christopher Strachey's draughts program showed that stored-program computers could do more than numerical calculation. It represented board positions, generated moves, evaluated alternatives, and attempted strategic play on early British computers, making game playing one of AI's first public testbeds.

## People & Place
- Christopher Strachey: Developer of the early draughts program

Key place: National Physical Laboratory

## Historical Background
In the early 1950s, running a non-numerical game program stretched tiny memories and fragile programming tools.

## Canonical Source
Logical or non-mathematical programmes. https://dl.acm.org/doi/10.1145/1455270.1455277

## Core Idea
The program encoded legal moves and board evaluation so the machine could choose actions rather than merely replay a script.

## Key Concepts
- Board Representation: A game state becomes data the computer can inspect.
- Move Generation: The program enumerates legal moves instead of following a fixed script.
- Evaluation: Candidate positions are scored so the machine can choose.

## Impact
It helped establish board games as controlled environments for search, evaluation, and machine intelligence demonstrations.
