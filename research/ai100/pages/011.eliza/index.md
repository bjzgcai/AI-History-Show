# ELIZA

## Full Name
ELIZA: A Computer Program for the Study of Natural Language Communication Between Man and Machine

## Year / Period
1966

## Type
ChatBot

## One-Sentence Summary
ELIZA is a major AI100 achievement connecting ChatBot to later AI systems.

## Hero Description
ELIZA showed how far simple language pattern matching could go in producing the feeling of conversation. Its DOCTOR script reflected user phrases back as questions, making the interaction memorable precisely because the mechanism was shallow. The program became a landmark in chatbot history and in debates about anthropomorphizing machines.

## People & Place
- Joseph Weizenbaum: Creator of ELIZA

Key place: MIT Artificial Intelligence Laboratory

## Historical Background
In the 1960s, AI language systems were mostly symbolic and hand-built. ELIZA tested whether a small set of scripts could sustain the impression of natural conversation.

## Canonical Source
ELIZA: A Computer Program for the Study of Natural Language Communication Between Man and Machine. Joseph Weizenbaum, Communications of the ACM, 1966. https://doi.org/10.1145/365153.365168

## Core Idea
The DOCTOR script does not understand therapy; it transforms recognized patterns into reflective prompts that invite the user to continue.

## Key Concepts
- Script: A script defines keywords and response templates for a conversational role.
- Reflection: ELIZA often turns a user statement back into a question using the user’s own words.
- ELIZA Effect: People may attribute understanding to a system that only follows simple conversational rules.

## Impact
ELIZA still matters because modern systems are also judged by interaction, expectation, and user projection, not only by internal mechanisms.
