# Historical Background

Before deep learning dominated AI, search was one of the field’s central metaphors. Holland brought biological adaptation into computation, making it possible to study learning and optimization in artificial populations.

# Core Idea

A genetic algorithm does not improve one solution directly. It evolves a population by repeatedly scoring, selecting, recombining, and mutating candidates.

# Key Concepts

- Population Search: The algorithm keeps many candidates at once, so search can explore multiple regions instead of following one path.
- Crossover: Parts of successful candidates are recombined, allowing useful partial structures to move into new solutions.
- Mutation: Random changes preserve diversity and let the population escape premature convergence.
